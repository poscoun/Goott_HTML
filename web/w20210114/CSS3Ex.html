CSS가 브라우저별로 부분적으로 다르게 구현될 수 있습니다. 그러므로 필요하다면 CSS 프로퍼티 앞에 브라우저를 식별할 수 있는 접두사를 붙여서 사용해야 함.

  1. -webkit- : 웹킷방식 브라우저용 (사파리, 크롬, ...)
  2. -moz- : 게코방식 브라우저용 (모질라, 파이어폭스...)
  3. -o- : 오페라 브라우저
  4. -ms- : 마이크로 소프트 인터넷 익스플로러
