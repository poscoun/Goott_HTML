HTML(HyperText Markup Language)

HyperText : 컴퓨터를 통하여 저장된 정보를 학습자(사용자)가 자신의 필요와 인지 (스타일)에 따라 자유롭게 검색하도록 도와주는 비순차적 텍스트 전개 원리

Markup (정보) : 문서 처리를 지원하기 위해 문서에 추가되는 정보 : 문서의 논리적 구성이나 체제와 같은 문서의 포멧을 지정하는 정보, 문서 내용 찾아보기 또는 찾아보기 작업(indexing) 방법을 지정하는 정보, 문서 내용 중의 요소와 같은 문서 또는 다른 문서 중의 다른 문서와 연결 정보 .. => 이런 마크업 정보를 표현하는 언어를 마크업 언어

특징

1. 태그(tag)로 이루어져 있다.
2. 형식(default) : <시작태그>   <./종료태그>
3. 웹브라우저에 출력되는 부분 :
4. <단독태그/>
5. 띄어쓰기, 줄바꿈(개행)은 직접 명령을 주어야 한다.
6. 대소문자를 구분하지 않는다.
7. 적당히 틀려도 브라우저가 적당히 실행해 준다.
