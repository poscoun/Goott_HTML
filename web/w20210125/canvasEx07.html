몇 가지 프로퍼티를 사용해서 화면에 표시하는 선 스타일 지정

lineWidth=value 선의 두께. 기본값 1.0 (양수만 사용 가능)
lineCap=type

선의 끝 모양
- butt : 기본값
- round : 선 너비의 1/2을 반지름으로 하는 반원
- square : 선 높이의 1/2인 선 양쪽 끝의 사각형

lineJoin=type

선과 선의 만남 (선의 교차점)
- round : 선과 선이 만나는 부분을 둥글게 처리
- bevel : 두 선 연결 부분에 단면을 만듦
- miter : 연결한 흔적없이 하나의 선처럼 만듦

miterLimit=value

lineJoin에서 miter type에 옵션 추가
연결 흔적 부분에 뾰족한 꼭지점이 생김 => 이 부분을 얼마나 없애 줄지 처리
