transition : 하나의 스타일에서 다른 스타일로 바뀌는 것

시간을 조절하는 기능을 더해서 애니메이션 효과를 만들어 낼 수 있음

transition : property || duration || timting-function
property : 애니메이션을 적용할 요소의 속성을 지정 ex) none, all, height...
duration : 트랜지션이 시작해서 끝날 때 까지의 시간을 초단위 지정 ex) 2s
timing-function : 속성의 중간값을 계산하는 방법을 정의하는 함수를 명시
- 1) ease : 천천히 시작 후 변환효과를 빨리하고 느리게 끝냄(기본설정)
- 2) ease-in : 느린시작
- 3) ease-out : 느린 끝
- 4) ease-in-out : 느린 시작, 느린 끝
- 5) linear : 시작부터 끝까지 동일한 속도
