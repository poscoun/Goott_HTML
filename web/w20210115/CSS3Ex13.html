그조적 Pseudo-Class

여러 개가 나열되어 있는 요소 중에서 특정 부분에만 다른 스타일을 적용

:nth-child(n) 항목 중에서 n번째 자식을 선택 (참고 : 수식으로도 가능 ex) 3n-1)
:nth-last-chile(n) 끝부터 n번째 자식을 선택
:nth-of-child(n) 유형이 같은 n번째 자식을 선택
:nth-last-of-child(n) 끝부터 유형이 같은 n번째 자식을 선택
:nth-child(ood) 홀수번째 유형을 선택
:nth-child(even) 짝수번째 유형을 선택

CSS2 부터 있던 구조적 Pseudo Class

:root 문서의 root 선택
:last-child 컨테이너 항목의 마지막 자식
:first-of-type 해당 유횽의 첫번째 항목을 선택
:last-of-type 해당 유형의 마지막 항목을 선택
:only-child 컨테이너 항목의 유일한 자식
:only-of-type 해당 유형을 가진 컨테이너 항목의 유일한 자식을 선택
:enpty 빈 항목을 선택 (주의. 어떤 텍스트도 들어있으면 안됨)
