    가상 클래스
    : 기본 선택자 외의 요소에 CSS 효과를 주고 싶을 때 사용
    : 특정한 상태 말고 규칙에 따라 상태가 바뀌는 요소를 선택할 때 사용

    기본문법
    선택자 : pesudo-class{속성:값;};

    많이 사용되는 요소

    pseudo-class

    요소 설명
    :link 링크변경
    :visited 한번 이상 방문한 링크의 색상을 변경
    :hover 마우스 커서가 링크 위에 올라갔을 때 변경
    :active 활성화된 요소에 대한 속성 지칭
    :focus 포커스를 받은 요소에 대한 속성 지정
    :frist-child 첫번째 자식에 대한 속성 지정
    :lang 특정 language에 대한 속성 지정(다국적 홈페이지에 유용
    :not 지정한 항복을 제외한 모든 항목을 선택
    :target 앵커를 이용해서 웹 페이지 안에서 타깃이 되는 부분에 표시 및 속성 지정 기능
